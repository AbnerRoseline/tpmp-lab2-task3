# Лабораторная работа №2, Задание 3: Система управления поездами

## Overview / Обзор

Этот проект представляет собой консольное приложение на языке C для управления данными о поездах. Он был создан в рамках выполнения Задания 3 Лабораторной работы №2.

Проект демонстрирует:

- Использование структур (`struct`) в C для представления данных.
- Разделение проекта на модули по модели "Клиент-Интерфейс-Сервер".
- Автоматизацию сборки проекта с помощью `Makefile`.
- Настройку непрерывной интеграции (CI) с использованием `GitHub Actions`.

## Usage / Использование

### Сборка проекта

Для компиляции проекта, находясь в корневом каталоге, выполните команду:

```bash
make
```

Эта команда создаст исполняемый файл `train_app` в корневой директории.

### Запуск приложения

Для запуска скомпилированного приложения выполните:

```bash
make run
```

Или напрямую:

```bash
./train_app
```

### Очистка

Чтобы удалить скомпилированные объектные файлы и исполняемый файл, используйте:

```bash
make clean
```

## Additional Notes / Дополнительные заметки

### Структура проекта (модель КИС)

- **Интерфейс**: `include/train.h`
  - Определяет структуру `TRAIN` и прототипы всех функций.
- **Сервер**: `src/train.c`
  - Содержит конкретную реализацию функций для работы с данными о поездах.
- **Клиент**: `src/main.c`
  - Главная программа, которая использует функции из "сервера" для демонстрации их работы и решения поставленных задач.
